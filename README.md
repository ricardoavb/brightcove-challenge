# Brightcove code challenge - Playwright tests

### How to run de project? :rocket:

1. Install npm dependecies

```bash
    npm install
```

2. Install browsers (Chromium, Firefox, Webkit)

```bash
    npx playwright install
```

3. Run the tests

```bash
    npm run test # headless
    npm run "test:headed"
```

4. Generate report

```bash
    npm run report
```
